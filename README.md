# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/shahid-shaikh-001/Neuronix.git
cd Neuronix
```

---

## 2️⃣ Setup Backend

```bash
cd Backend
npm install
```

Create `.env` file inside Backend folder:

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Run backend server:

```bash
npm run dev
```

---

## 3️⃣ Setup Frontend

```bash
cd Frontend
npm install
npm run dev
```
